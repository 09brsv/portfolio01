Meu start no mundo dev web foi em abril de 2021, quando desenvolvi um sistema básico de vendas com VBA pelo youtube.

Em dezembro de 2021, foi quando comecei a estudar focado em front end. Abaixo você verá melhor o que eu consigo fazer até aqui desde então.

Tenho disponibilidade de participar de processos seletivos para cursos e/ou entrevistas que me possibilitem a chance de conseguir meu primeiro emprego na área.

Busco participar de projetos simples ou mais complexos para adquirir experiência.

Html5

Entendo que Html é uma linguagem muito importante, seja para o usuário da página, como para quem for dar manutenção. Desta forma procuro cada vez mais deixar a marcação dos meus projetos com um código bem identado e limpo.

Css3

Em cada projeto que faço, procuro aprender e inserir novos estilos. Gosto muito de ver o que se pode fazer com CSS3. Estou empenhado em escrever um estilo que não seja desajeitado

Javascript

Percebi usando Javascript que no início parece muito difícil de aprender, mas praticando e estudando, rapidamente pega o jeito. É a linguagem de programação mais usada no mundo e não é atoa pois é muito diversa e intuitiva.

ReactJs

Git
